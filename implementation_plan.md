# Portfolio Implementation Plan - Vetrivel Murugan P

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: GSAP (Split Text effect), Framer Motion (Transitions, Hover), Lenis (Smooth Scroll)
- **Visuals**: Three.js (WebGL Particle Background), Lucide React (Icons)

## Implemented Sections
1.  **Hero**: Cinematic entrance with Three.js particles, GSAP split-text reveal, and typewriter subtitle.
2.  **About**: Personal bio with interactive detail cards.
3.  **Journey (Education & Experience)**: Custom staggered timeline showing internship at IPCS Global, NxtWave Academy, and College details.
4.  **Skills**: Animated grid with category-specific hover glows and progress indicators.
5.  **Projects**: 3D Tilt interactive cards for Dental Care, College ERP, and Resume Builder.
6.  **Certificates**: Clean grid layout with 10 verified links and award icons.
7.  **Contact**: Modern form UI with success state and social ecosystem links.
8.  **Design System**: Dark luxury theme (Charcoal/Black), Glassmorphism, Noise grain overlay, Custom branding font (Poppins/Inter).
